sfcurve

\(n \times n\) space-filling curves.

Basic

软件包 链接
Language R, C++
CRAN https://cran.r-project.org/package=sfcurve
GitHub https://github.com/jokergoo/sfcurve

Publication Zuguang Gu, Construction, Transformation and Structures of 2x2 Space-Filling Curves. arXiv 2024.

Example

2x2 curve

library(sfcurve)
p = sfc_2x2("I", "2222", rot = 270)
p
